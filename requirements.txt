# CRE Underwriter — Python dependencies
#
# The underwriting engine (Phase 1-2) and SQLite storage (Phase 3) use only the
# Python standard library. The dashboard (Phase 4) adds Streamlit; the RentCast
# integration (Phase 5) adds requests. AI document extraction (Phase D) reuses
# requests to call the Anthropic API — no new dependency.
#
# Install with:  pip install -r requirements.txt

streamlit>=1.42          # Phase 4: dashboard UI (>=1.42 for native OIDC st.login/st.user)
requests>=2.31           # Phase 5: RentCast client + Phase D: Anthropic API client
fpdf2>=2.7               # Phase E: render the deal memo to a one-page PDF
Authlib>=1.3.2           # Phase F: Google sign-in (required by Streamlit's st.login OIDC)
psycopg2-binary>=2.9     # Phase G: PostgreSQL backend (used only when DATABASE_URL is set)

# Phase D needs an Anthropic API key in the ANTHROPIC_API_KEY environment variable
# (https://console.anthropic.com). Without it, manual entry still works; AI import
# just shows a friendly "set the key" notice.
#
# Optional: reading EXCEL uploads (.xlsx/.xls) needs openpyxl (pip install openpyxl).
# Without it, PDF / image / CSV imports still work; Excel shows a "export to CSV" note.
#
# Phase F (Google sign-in) reads OIDC credentials from .streamlit/secrets.toml (an
# [auth] section). That file holds secrets and is gitignored — never commit it.
